Qué incluye

Combinamos NOC (disponibilidad) + SOC (seguridad) + protección de correo para cubrir riesgos reales en operación.

NOC 24×7

  • Disponibilidad / HTTP / TLS / DNS / Puertos
  • Alertas por correo / Teams / Slack
  • Reporte mensual de desempeño

SOC (MDR)

  • EDR + correlación de eventos
  • Casos de uso: ransomware / C2 / movimiento lateral
  • Playbooks y respuesta guiada

Seguridad de correo

  • Controles de postura SPF / DMARC / DKIM
  • Autenticación avanzada
  • Concientización (opcional)
